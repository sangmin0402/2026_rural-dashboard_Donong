## 1. 프로젝트 개요

**법적 배경**  
농촌공간 재구조화 및 재생지원에 관한 법률(2023) 시행에 따라 경기도 내 농촌 시군은 각자 기본계획을 수립해야 함. 이 플랫폼은 계획 수립을 지원하기 위한 데이터 시각화 도구임.

**대상 시군** (15개)  
남양주시, 가평군, 양평군, 여주시, 이천시, 안성시, 양주시, 포천시, 연천군, 파주시, 광주시, 화성시, 평택시, 용인시, 김포시

**수업**: 지역활성화시스템론 (2026)  
**담당**: 주영 · 선혁 · 유하 (남양주시 케이스)

## 2. 주요 기능

### 🌐 랜딩 포털 (첫 화면)
- 플랫폼 소개 및 농촌다움 개념 설명
- 경기도 SVG 미니맵 — 시군 클릭 시 삶터·일터·쉼터 지표 요약 패널
- KPI 카운터 (15개 시군 · 19개 지표)
- 5대 권역 소개 (보고서 표 2-36 기준)
- 브라우저 뒤로가기로 첫 화면 복귀 가능

### 🗺️ 지표 지도
- Leaflet.js 기반 코로플레스 지도
- 지표 선택 드롭다운으로 실시간 색상 갱신
- 줌인 시 읍면동 레이어 자동 전환
- 시군 클릭 → 우측 사이드 패널에 상세 지표 표시
- 5분위(quantile) 기반 색상 분류

### 📊 레이더 차트
- 선택 시군의 삶터·일터·쉼터 종합점수 시각화
- 삶터/일터/쉼터/시나리오 탭 전환

### ⚖️ 시군 비교
- 최대 5개 시군 동시 선택
- 지표별 막대 차트 비교
- 시군 추가/제거 토글

### 🔮 시나리오 분석
- 정책 레버 5종 슬라이더 조정
- before/after 레이더 차트 비교
- 적용 가능 지표: L4, W6, W7, R4, R6

### 🔍 시군 검색
- 헤더 검색창으로 시군 이름 검색
- 드롭다운 자동완성

## 8. 실제 데이터 반영 가이드

### 데이터 교체 위치: `js/app.js` → `CITIES` 객체

```javascript
// 예시: 남양주시 L1(인구증가율) 실제값으로 교체
namyangju: {
  indicators: {
    L1: 0.8,   // ← 이 값을 실제 통계로 교체
    ...
  }
}
```

### 교체 순서 권장

1. **공통지표 우선** — L1~L4, W1~W4, R1~R3 (11개 × 15개 시군 = 165개 값)
2. **남양주 자율지표** — L5~L6, W5~W7, R4~R6 (8개)
3. GIS 분석 필요 지표 (R1, R2, R3, L4) 는 별도 처리 후 삽입

### GeoJSON 관련 주의사항

> ⚠️ `gyeonggi-sigun.geojson`의 `properties.name` 필드는 이중 이스케이프 상태  
> 시군 이름이 필요할 때는 반드시 `CITIES[id].name` 사용

```javascript
// ❌ 잘못된 방법
const name = feature.properties.name; // 깨진 문자열

// ✅ 올바른 방법
const cityId = feature.properties.id;
const name = CITIES[cityId].name;
```
